基于 React18 + TS + Vite 开发的 OBI 工具，可快速生成报表，数据大屏。

## 安装和使用

```
// clone项目到本地
git clone https://github.com/ouyanghao0901/datao.git

// 打开目录
cd datao

// 安装依赖
npm i

// 运行
npm run dev
```
